# Grocerease

Grocerease is an Ionic + Vue grocery list app.

## Development

- Install dependencies: `npm install`
- Run dev server: `npm run serve`
- Build production bundle: `npm run build`
- Run unit tests: `npm run test:unit`
- Run e2e tests: `npm run test:e2e`
- Run linter: `npm run lint`

## Quick start tutorial

1. Open the app and go to the **Items** tab.
2. Tap the center **Add** button to create your first item.
3. Tap an item name to edit it (rename, move parent, add sub-items, add a note).
4. Use the checkbox in **Items** to mark what you need to buy.
5. Switch to **Shopping list** to see all checked items.
6. In **Shopping list**, tick an item to mark it done and remove it from the active list.
7. Use the **Add** button in **Shopping list** to add one-time items.
8. Open **Settings** (top-right icon) to change sorting, enable **History**, and share the current shopping list.

### Tips

- In **Items**, swipe right on a row to quickly add a note.
- In **Items**, swipe left on a row to delete an item (and its children).
- Use the **Search** field in **Items** to filter by item name.
- If **History** is enabled, completed items can be restored from the **History** section in **Shopping list**.
